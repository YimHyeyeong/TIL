  * 문자열 수식 계산의 일반적인 방법

    * step1 중위 표기법의 수식을 후위 표기법으로 변경한다(스택 이용)

      ```
      중위 표기법
      - 연산자를 피연산자의 가운데 표기하는 방법 ex) A+B
      후위 표기법
      - 연산자를 피연산자 뒤에 표기하는 방법
      ex) AB+
      ```
